# React Form Basics

This project demonstrates how to create and handle forms in React using
controlled components.

## Concepts Covered
- useState
- Controlled Inputs
- onChange & onSubmit
- Basic Form Validation
- Form Reset

## Form Fields
- Name
- Email
- Password
- Gender (Radio Buttons)
- Accept Terms (Checkbox)

## Validation Rules
- All fields are required
- Email must include '@'
- Password must be at least 6 characters

## Tech Stack
- React
- JavaScript
- CSS

## How to Run
1. Clone the repository
2. Run `npm install`
3. Run `npm start`
